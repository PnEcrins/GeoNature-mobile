## Action bar icon

### Dark theme

* size: default Action Bar icon size
* padding: 0
* color: white, opacity: 80%
* color disabled: white, opacity: 50%

### Light theme

* size: default Action Bar icon size
* padding: 0
* color: black, opacity: 80%
* color disabled: black, opacity: 50%
